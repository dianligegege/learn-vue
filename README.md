# learn-vue

## Project setup
``` html
yarn install
```

### Compiles and hot-reloads for development
```html
yarn run serve
```

### Compiles and minifies for production
```html
yarn run build
```

### Run your tests
```html
yarn run test
```

### Lints and fixes files
```html
yarn run lint
```
